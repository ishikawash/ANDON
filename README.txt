行燈(ANDON) is private experimental ray tracer.== prerequisites* gcc 4.2.1 or later* boost 1.47* OpenGL Mathematics 0.9.2.7  http://glm.g-truc.net/* Open CTM 1.0.3   http://openctm.sourceforge.net/== features* Blinn–Phong shading model* Bounding Volume Hierarchy* Area lighting* Open CTM model format support== compiling & running$ cd src && make && make runthe program will generate .ppm file (out.ppm).
